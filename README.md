我们先简单介绍下我们要实现的应用的功能：
日记本有引导页，具备列表显示功能、日记的新建、查看阅读、编辑功能，并加入了搜索功能。日记的数据存储的本地手机数据库内，提供3个页面进行切换。

1.首先是引导页：引导页作为应用的一个基本的功能，会为大家实现并提供解决方案。

2.应用页面基本框架：包括一个底部导航栏，带三个页面切换，并在切换时保持页面数据状态。

3.日记的增删改查功能：存储在手机自带数据库中。

4.日记的搜索功能。

5.关于页面。

6.列表时时通知更新功能。

7.复制剪贴板功能等。
